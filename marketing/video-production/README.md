# MnemoPay Video Ad Production Pipeline

## First Ad: "The Math" (60 seconds)

### Tools You Need (all free)
1. **ElevenLabs** — voice clone + voiceover (you have API key)
2. **Hedra** — photo → talking head (sign up free at hedra.com)
3. **Kling AI** — cinematic B-roll (you have account)
4. **Suno** — music bed (you have account)
5. **CapCut** — final edit + captions (free, no watermark)

### Quick Start
```bash
# Step 1: Generate voiceover
node generate-voiceover.js

# Step 2: Follow the shot list in SHOT-LIST.md
# Step 3: Edit in CapCut using the assembly guide
```
