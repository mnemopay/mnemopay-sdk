# MnemoPay Focus — 2026-04-09 → 2026-06-09

**Principle:** engineering is not the bottleneck. Paying customers are. Everything that isn't directly producing a pilot conversation gets paused for 60 days.

---

## What we're doing

1. **Wedge: Lightning + MCP** (public) — 5 hand-written DMs/day to MCP server authors. First 10 adopters get free-forever tier.
2. **Wedge: Paystack / Africa** (private) — 3 LinkedIn DMs/day to African SaaS & fintech founders. $500/mo design-partner pilot.

5. **Weekly review** — every Friday, in `gtm/outreach.md`, ruthless honest count of who got contacted, who replied, who's in pilot.

## What we're NOT doing for 60 days

These are killed, not paused. "Paused" becomes "still running at low intensity" which becomes a distraction.

- ❌ **Dev.to tutorials** — reaches builders, not buyers. One more post allowed only if it directly targets MCP authors.
- ❌ **Show HN / Hacker News launches** — wrong audience for buyer conversion.
- ❌ **Twitter / X autopost pipeline** — already broken (API credits depleted). Stays off.
- ❌ **LinkedIn auto-posting** — hand-written only. Quality over cadence.
- ❌ **YouTube video series** — except the 1-minute calculator walkthrough, which is outreach ammunition.
- ❌ **New Smithery / Fleek / etc integrations** — unless a paying pilot asks for it.
- ❌ **Agent framework PRs** (CrewAI, LangGraph, etc) — already marked dead in memory. Stays dead.
- ❌ **Blog posts about architecture** — nobody buys software because the architecture is cool.
- ❌ **Comparison pages against Mem0 / Kite / Skyfire** — existing pages stay, no new ones.
- ❌ **New module development** — no new FICO signals, no new behavioral engines, no new memory strategies. Freeze the surface area.
